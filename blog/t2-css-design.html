CSS Positioning Cheat Sheet

Posted on October 18, 2014

What are the differences between Margin, Border, and Padding?

The margin encloses the border, which encloses the padding, which encloses the content. You can use CSS to style all three of these independently.

What is Content?

This is the area that you will be styling. If you dont apply any style to it, it's default size will be just large enough to contain the elements within it. In this example the total width of the element is 100px

What is Padding?

The padding clears an area around the content (inside the border) of an element. Padding is the horizontal and vertical space that is set around the content area of the targeted element. Padding is on the inside of a box, not the outside. In this example the total width of the element is 140px.

What are Borders?

Borders fit in between the padding and the margins. You can apply borders to any visible element you like: divs, lists, and so on. Most likely you will be surrounding blocks of text and images. Be careful with borders as they DO add width to the element even if it's only 5 pixels. In this example the total width of the element is 150px.

What Are Margins?

Margins in CSS make up the vertical and horizontal areas between elements. If elements have no margins around them, they will bump right up against each other. Some elements have margins by default, so often even if you haven't added a margin, you may still see one. In this example the total width of the element is 450px.
